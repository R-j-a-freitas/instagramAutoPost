Instagram Auto Post

Automação de Marketing para Instagram

Apresentação comercial — Plataforma de publicação automática integrada com Google Sheets e IA

O desafio das empresas

  • Tempo — Publicar no Instagram consome horas por semana
  • Consistência — Manter um calendário editorial é difícil
  • Escala — Uma pessoa não consegue gerir múltiplas contas
  • Engajamento — Responder a comentários manualmente é moroso

A solução

Instagram Auto Post — Plataforma que publica posts, Stories e Reels de forma programada.

  • API oficial do Instagram (sem risco de bloqueio)
  • Publicação automática na hora agendada
  • Geração de conteúdo com IA (Gemini)
  • Autoresposta a comentários

Fluxo de trabalho

Google Sheet (calendário) → App verifica → Publica no Instagram
Data, Hora, Texto, Imagem → Post pronto? → Feed, Story, Reel

Uma única fonte de verdade — O Google Sheet é o calendário. A equipa edita; a app publica.

Instalação rápida

Windows: Duplo clique em install.bat → depois run.bat

Linux:

chmod +x install.sh run.sh run_autopublish.sh
./install.sh
./run.sh

A app abre em http://localhost:8502. Configura o .env e liga Google + Instagram na secção Configuração.

Publicação no servidor

  • Apresentação HTML — Serve docs/apresentacao.html via nginx ou outro servidor web
  • App Streamlit — Instala no servidor com install.sh, corre com run.sh ou como serviço systemd
  • Media — Pasta partilhada (Samba) + nginx para servir ficheiros (ver docs/SAMBA_E_TESTES.md)
  • Autopublish 24/7 — Cron (Linux) ou Task Scheduler (Windows) com run_autopublish.sh / run_autopublish.bat
